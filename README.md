## Overview
A front-end e-commerce website built using HTML and CSS, focused on user interface design, layout structure, and responsive behavior. This project simulates a real-world online shopping experience and demonstrates the ability to build and troubleshoot UI components.

## Technologies
- HTML
- CSS

## Features
- Product listing layout
- Navigation menu
- Structured page design for usability
- Responsive layout for different screen sizes

- Designed UI components and improved visual consistency
- Debugged layout and spacing issues
- Improved navigation and user experience
